# 🔄 更新网站内容

## 问题说明

腾讯云 CloudBase 目前**不会自动同步**GitHub 的更新，需要手动重新部署。

---

## 方法一：腾讯云 CloudBase 控制台重新部署（推荐）

### 步骤：

1. **访问腾讯云 CloudBase 控制台**
   - 网址：https://console.cloud.tencent.com/tcb
   - 登录你的腾讯云平台

2. **找到你的环境**
   - 环境 ID：`geogebra-env-6g8j4ayb041b3d52`
   - 点击进入环境详情

3. **重新部署**
   - 点击「托管」→「静态网站」
   - 找到 `geogebra-tutorial` 文件夹
   - 点击「重新部署」或「更新」
   - 等待部署完成（约 1-2 分钟）

4. **验证更新**
   - 访问：https://geogebra-env-6g8j4ayb041b3d52-1412389188.tcloudbaseapp.com/geogebra-tutorial/
   - 按 `Ctrl+Shift+R` 强制刷新清除缓存
   - 检查错误弹窗是否消失

## 方法二：使用 Vercel 自动部署（一劳永逸）⭐

**强烈建议迁移到 Vercel**，因为：
- ✅ 代码推送到 GitHub 后**自动部署**
- ✅ 无需手动操作
- ✅ 全球 CDN 加速
- ✅ 完全免费

### 部署步骤：

1. **访问 Vercel**
   - 网址：https://vercel.com/new
   - 使用 GitHub 账号登录

2. **导入项目**
   - 选择 `geogebra-tutorial` 仓库
   - 点击「Deploy」

3. **完成**
   - 30 秒后获得永久网址
   - 格式：`https://geogebra-tutorial-xxx.vercel.app`
   - 以后每次 `git push` 都会自动更新

---

## 方法三：使用 Netlify 自动部署

### 步骤：

1. **访问 Netlify**
   - 网址：https://app.netlify.com/drop

2. **拖拽部署**
   - 将整个 `geogebra-tutorial` 文件夹拖入页面
   - 等待上传完成

3. **完成**
   - 获得永久网址
   - 格式：`https://xxx-xxx.netlify.app`
