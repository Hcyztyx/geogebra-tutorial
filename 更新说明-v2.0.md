## ✨ 新增功能

### 1. 🏆 完善的评分系统

#### 评分机制
- **基础分**: 每个任务有固定分值（10-300 分）
- **尝试次数影响**: 
  - 第 1 次尝试：100% 得分
  - 第 2 次尝试：80% 得分
  - 第 3 次尝试：60% 得分
  - 最低保障：50% 得分
- **提示惩罚**: 每次使用提示扣除 10% 得分

#### 分数计算示例
```
任务基础分：100 分
- 第 1 次通过 + 无提示 = 100 分 ✅
- 第 2 次通过 + 无提示 = 80 分
- 第 1 次通过 + 1 个提示 = 90 分
- 第 3 次通过 + 2 个提示 = 48 分
```
